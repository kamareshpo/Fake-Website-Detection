# Fake-Website-Detection
AI-powered Chrome extension to detect fake and malicious websites in real-time, helping users browse safely with trust scoring and analysis.

Features
Real-time Website Analysis
Automatically analyzes the current website URL
AI-based Detection
Evaluates websites using multiple trust factors
Trust Score System
Displays a score indicating how safe a website is
Fake Website Alerts
Warns users about suspicious or dangerous sites
Works on All Websites
Supports both HTTP and HTTPS URLs
User-friendly Interface
Clean popup UI with clear results and indicators
